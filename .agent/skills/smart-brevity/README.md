# Skill: smart-brevity

Rewrites messages applying "Smart Brevity" principles — cuts the unnecessary, leads with what matters, structures for quick scanning. Output is always in pt-BR.

## Usage

Send via Telegram:

```
Reescreve essa mensagem: <paste text>
```

Or variations like "resume isso", "encurta esse texto", "melhora essa mensagem", "smart brevity: <text>".

## Prerequisites

None. Pure text skill, no external dependencies.
