04 — Etapas
$ /superpowers:brainstorm

Brainstorm

O agente:

  • Faz perguntas
  • Define escopo
  • Escolhe stack
  • Cria uma spec clara

Use quando:

  • Início de qualquer feature
  • Dúvida de abordagem
  • Refatorações

Aqui você NÃO escreve código — você define o quê e como.

04 — Etapas
$ /superpowers:write-plan

Write Plan

O agente:

  • Quebra em tarefas pequenas
  • Define ordem de execução
  • Sugere estrutura de arquivos
  • Define critérios de validação

Muito útil para:

  • Não se perder durante a execução
  • Organizar execução incremental
  • Critérios de aceite claros
04 — Etapas
$ /superpowers:execute-plan

Execute Plan

O agente:

  • Implementa passo a passo
  • Cria subagentes em paralelo
  • Aplica TDD quando possível
  • Faz validações e code review

Resultado:

  • Velocidade de execução
  • Consistência de código
  • Menos erros e retrabalho

05 — Fluxo recomendado

Boas Práticas do Fluxo

  • Não pular etapas — revisar antes de avançar
  • Usar nova sessão entre etapas
  • Cada etapa gera um .md — reutilize como contexto
  • Passe o markdown anterior ao iniciar a próxima etapa
