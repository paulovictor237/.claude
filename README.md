# Claude Code Boillerplate

## How to use

### Linux / macOS

With curl:

```sh
curl -fsSL https://raw.githubusercontent.com/paulovictor237/claudinho/main/install.sh | bash
```

With wget:

```sh
wget -qO- https://raw.githubusercontent.com/paulovictor237/claudinho/main/install.sh | bash
```

Download and execute:

```sh
curl -fsSL https://raw.githubusercontent.com/paulovictor237/claudinho/main/install.sh -o /tmp/install.sh

bash /tmp/install.sh
```
